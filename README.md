# db-ntua
Project for class SQL Databases @ece-NTUA 2020, simple CMS App  

Basic Express-Pug Application

git clone https://github.com/christos97/db-ntua &&
cd db_ntua &&
npm install && npm start

Γενικά φτιαξε το db config και παιζει να σου πεταξει ερρορ η βαση , κανε alter user απο το cli της mysql

localhost:3000/ einai o server

Ftiakse ena table admins me fields email,password kai enan admin me admin@admin.com, adminpass

kai dokimase na kaneis login 

Genika allazeis branches me git checkout [branch-name] kai vlepeis kai an eisai pisw i mprosta se commits

git fetch -> paei to head sto teleutaio commit tou branch p vriskesai (xrisimo)

git pull -> kanei git fetch && git merge (genika to merge kanei kati perierga pragmata sto local repo sou px tha s leei mesa

ston editor ti thes na krathseis kai ti oxi kai kati tetoies malakies)

an thes to branch mou opws einai kane

git checkout master

git fetch

git reset --hard origin/master

kai tha kanei overwrite ta panta (tha pareis genika hard copy tou master branch opote tha einai identical)

afou loipon kaneis o,ti einai na kaneis kai thes na pushareis

git add .

git commit -m 'message'

git fetch (gia kalo kai gia kako, mallon den xreiazetai)

git push -u origin/giannis

an den se afhsei min kaneis pote -f, apla kane git pull kai meta pali push
